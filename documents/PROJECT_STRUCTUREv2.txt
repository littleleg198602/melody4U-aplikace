📁 PROJECT_STRUCTURE.md

(aktuální stav projektu Melody4U – leden 2026)

1️⃣ Přehled architektury (high-level)

Projekt Melody4U je rozdělen do tří hlavních částí:

FORPSI hosting – statický výstup React aplikace (produkce)

Frontend (React + Vite) – vývojový zdroják UI

Backend API (Node.js / Express) – logika, Stripe, audio, DB, storage

Každá část má jasnou roli a nesmí se míchat.

Význam

FORPSI web = pouze public UI

Nesmí obsahovat business logiku

Neřeší Stripe ani DB

🔒 Pravidlo SPA routingu na FORPSI (HashRouter)

FORPSI je statický hosting bez server-side rewrite → přímé URL jako /create vrací 404. 

PROJECT_STRUCTUREv1

Produkce musí používat HashRouter routing:

správně: /#/create, /#/p/<id>

špatně: /create, /p/<id>

V UI je zakázáno používat <a href="/create"> nebo window.location="/create".
Používej pouze:

<Link to="/create"> (React Router), nebo

window.location.href="/#/create" (jen v krajním případě).

🧨 POZNÁMKY (tohle si tam přidej hned pod strukturu)
1) Pravidlo: jen jeden CreatePage „zdroj pravdy“

src/pages/create/CreatePage.jsx je jediný CreatePage pro produkci.

src/pages/CreatePage.jsx nesmí existovat (jinak CI/build často importuje špatný soubor).

2) Zakázané “kopie” soubory v src/pages/

CreatePage - kopie*.jsx musí pryč (nebo do src/_archive/ mimo build).

Jinak se dřív nebo později stane:

špatný import v App.jsx

nebo build error typu “default is not exported”.

3) App.jsx import (kritické)

App importuje Create jen takhle:

import CreatePage from "./pages/create/CreatePage.jsx";

(aby to na hostingu bez server-side routes nepadalo)

- Komponenty homepage jsou izolované (UI-only) a nesmí zasahovat do CORE logiky.
- Demo MP3 nejsou v repozitáři, ale hostují se na FORPSI v /www/demo.

Z tohoto důvodu nebyla struktura projektu aktualizována, pouze byla změna zaznamenána v PROJECT_HISTORY.

## Právní a compliance vrstva (Legal & Trust)

Projekt Melody4U obsahuje povinné právní stránky z důvodu:
- práce s osobními údaji (uživatelé, e-maily)
- přihlášení / účtů
- plateb (Stripe)
- cookies a analytiky

Tyto stránky jsou:
- statické obsahové stránky
- nezasahují do CORE, FLOW ani backend logiky
- jsou přístupné pouze přes footer

Povinné stránky:
- /terms – Obchodní podmínky
- /privacy – Ochrana osobních údajů (GDPR)
- /cookies – Informace o cookies

Pravidla:
- žádná z těchto stránek nesmí volat API
- žádná z těchto stránek nesmí měnit stav aplikace
- slouží výhradně k právnímu a důvěrovému krytí projektu
