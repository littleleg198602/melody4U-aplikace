# Melody4U – Documents

Tato složka je zdroj pravdy pro provozní postupy projektu.

## Pravidlo pro další změny
- Při každé změně build/deploy procesu se musí aktualizovat i dokumentace v této složce.
- Minimálně se aktualizuje `PROJECT_FLOW_UPDATED.txt`.

## Co nově dokumentujeme
- Build/deploy pravidla pro Forpsi.
- Výkonové chování načítání knihovny hudby (Create krok 2).
- Cookie consent povinnosti (zobrazení lišty při první návštěvě).
- Navigace Ukázky + jazykové verze (CZ/EN) a jejich routy.
